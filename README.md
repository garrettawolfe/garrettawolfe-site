# Garrett Wolfe - Personal Website

Modern personal website showcasing professional experience, interests, and current projects.

## Setup

This is a static website built with vanilla HTML, CSS, and JavaScript, optimized for GitHub Pages hosting.

## Local Development

Simply open `index.html` in a web browser, or use a local server:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js (with http-server installed)
npx http-server
```

Then visit `http://localhost:8000` in your browser.

## Deployment to GitHub Pages

1. Push all files to your GitHub repository: `https://github.com/garrettawolfe/garrettawolfe-site`
2. Go to repository Settings > Pages
3. Select the branch (main) and folder (root)
4. Your site will be available at `https://garrettawolfe.github.io/garrettawolfe-site/`

## Adding Assets

- **Company Logos**: Add to `assets/images/logos/` (PNG or SVG recommended)
- **Photos**: Add to `assets/images/photos/` (JPG or PNG recommended)
- **Background Music**: Add to `assets/audio/` (MP3 or OGG format recommended)

## Customization

- Update social media links in `index.html`
- Modify color scheme in `styles/main.css` (CSS custom properties)
- Add/update content in `index.html` sections
